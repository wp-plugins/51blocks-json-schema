== Description ==

51Blocks JSON Schema Creator allows business owners to add schema markup to a single page or entire site without outputting the markup on the actual site.  This tool also makes it very easy to validate your markup against Google Structure data testing tool  

Schema is the result of collaboration between Google, Bing, Yandex, and Yahoo! to help you provide the information their search engines need to understand your content and provide the best search results possible at this time.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `json-schema-generator` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
